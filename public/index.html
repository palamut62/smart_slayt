Yardım & Kullanım

smart.slayt; bir konuyu webden araştırıp 1080×1350 (4:5) bilgi kartı serisine dönüştürür. Aşağıda sağlayıcılar, modeller, özellikler ve nasıl kullanılacağı var.

🚀 Nasıl Kullanılır

  1. Ayarlar'dan bir sağlayıcı seç (OpenRouter için API anahtarı gir; Codex için makinede codex login yeterli).
  2. Stüdyo'da bir konu yaz (ör. "Bun runtime kurulumu").
  3. Dil, adım sayısı, şablon ve renk paleti seç.
  4. İstersen Derin araştırma'yı aç (her başlık için ayrı web araması — daha kapsamlı, biraz yavaş).
  5. Kartları Üret'e bas. Üretilen seri otomatik olarak Kütüphane'ne kaydedilir.
  6. Kartları tek tek veya ZIP olarak indir, ya da 𝕏'te paylaş.

Sağlayıcılar

🌐 OpenRouter

Web araştırması Perplexity Sonar ile yapılır, kart metinleri seçtiğin içerik modeliyle yazılır. sk-or-v1-... API anahtarı gerekir; kullanım başına kredi harcar.

Modeller: DeepSeek V4 Pro/Flash, Claude Sonnet 4.5, GPT-4o, Gemini 2.5 Flash. Araştırma: Perplexity Sonar / Sonar Pro.

⚡ Codex CLI

Makinedeki codex CLI'ı kullanır. Web araştırmasını kendisi yapar (tek ajan çağrısı). Ayrı API anahtarı gerekmez — mevcut ChatGPT oturumunu (codex login) otomatik kullanır.

Model alanı boş bırakılırsa Codex'in varsayılan modeli kullanılır (ör. gpt-5.3-codex).

⚠️ Otomatik yedek: Sağlayıcı Codex seçiliyken bağlantı/oturum yoksa veya Codex hata verirse, uygulama otomatik olarak OpenRouter'a geçer (anahtar tanımlıysa) ve bir uyarı gösterir. Üretim sonrası "hazır · ⚡ Codex / 🌐 OpenRouter" etiketi hangi sağlayıcının kullanıldığını belirtir.

Özellikler

  • 5 şablon: Editorial, Bold, Minimal, Scrapbook, Terminal — kartların yerleşim/tipografi derisi.
  • 7 renk paleti: Kraft, Forest, Midnight, Blush, Ocean, Sunset, Noir.
  • Derin araştırma: her alt-başlık için ayrı web araması yapıp daha doyurucu içerik üretir.
  • Otomatik logolar: bilinen marka/araçlar için gerçek logolar webden çekilip karta yerleştirilir.
  • Kütüphane: her seri saklanır; carousel'de görüntüle, ZIP indir veya sil.
  • 𝕏 paylaşımı: seriyi tek tweet ya da thread olarak paylaş (Ayarlar'dan OAuth 1.0a anahtarları).
  • Açık/Koyu tema ve 8 dil desteği (TR, EN, DE, FR, ES, IT, RU, AR).
